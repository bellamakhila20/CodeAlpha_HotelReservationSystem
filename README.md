# Hotel Reservation System

## Description

Hotel Reservation System is a Java-based console application that allows users to view available rooms, book rooms, cancel reservations, and manage hotel room bookings through a simple menu-driven interface.

## Features

- View available rooms
- Book rooms
- Cancel booking
- Display booking status
- Menu-driven interface
- Object-Oriented Programming

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner Class

## How to Run

Compile:

```bash
javac HotelReservationSystem.java
```

Run:

```bash
java HotelReservationSystem
```
